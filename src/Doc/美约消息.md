## 美约消息类型 ##

>这里的自定义消息指的是网易type 为100 下 我们自己的消息类型
>
>即在type =100时，我们自己在body里面定义的消息类型用　msg_type 标志  

### 消息类型 ###

1.

    common   常规类型自定义消息  后半部分带链接 前部分为消息内容

	{
	  "msg_type":"common",
	  "msg_body":"这是条测试消息",
	  "msg_link":"/order/1",
	  "msg_link_text":"查看详情"		
    }

2.
    
	finish_prepay  完成预约金  前部分有特殊前缀  中间为消息内容  后部分为链接

	{
	  "msg_prefix":"[约吃饭]"
	  "msg_type":"finish_prepay",
	  "msg_body":"这是条测试消息",
	  "msg_link":"/order/1",
	  "msg__link_text":"查看详情"	
    }


> 目前就这2种类型
